# MammoAssist

MammoAssist is a multimodal breast cancer analysis project that combines ultrasound image features, EHR features, and rule-based triage analysis.

## Included files

- `mammoassist-classification.ipynb` - multimodal classification pipeline
- `mammoassist-segmentation.ipynb` - segmentation workflow
- `mammoassist-ehr.ipynb` - EHR processing and modeling notebook
- `statistical_analysis.py` - statistical analysis utilities and tests
- `global_splits.json` - precomputed global train/validation/test splits

## Notes

- The repository currently includes the main notebooks and analysis script used for the project.
- Large datasets, outputs, and experimental artifacts are intentionally kept out of version control.

## Usage

Open the notebooks in Jupyter or VS Code and run them in order if you want to reproduce the pipeline.
